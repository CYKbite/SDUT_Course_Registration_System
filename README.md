# 🚀 山东理工教务系统自动抢课脚本 (Edge版)

> **⚠️ 特别注意：**
> 由于目前学校选课系统未开放，当前脚本仅完成了**登录**并**进入选课页面**阶段。无法进行真实选课/抢课操作。
> 后续的抢课功能需在系统开放后，根据实际界面逻辑完成剩余调试。

这是一个基于 Python 的 **Selenium** 自动化工具。它模拟人操作浏览器，用于自动登录山东理工大学正方教务系统。

## 🛠️ 1. 环境准备
在运行脚本之前，请确保你的电脑已安装以下软件：

* **Python 3.10+**: [下载并安装](https://www.python.org/downloads/) (安装时务必勾选 **"Add Python to PATH"**)
* **Microsoft Edge 浏览器**: [下载并安装](https://www.microsoft.com/edge)

---

## 📦 2. 依赖安装

### 第一步：获取项目
克隆项目到本地并进入目录：
```bash
git clone <你的GitHub仓库地址>
cd <项目目录>

### 第二步：配置虚拟环境 (推荐)

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境 (Windows)
.\venv\Scripts\activate
```

### 第三步：安装依赖库

使用 pip 安装 Selenium 和 WebDriver Manager：

```bash
pip install selenium webdriver-manager
```
如果命令失效请去官网根据edge版本下载 webdriver-manager

---

## ⚙️ 3. 配置与驱动

### 关键组件：驱动程序

本脚本使用了 `webdriver_manager` 库，它会自动检测你的 Edge 浏览器版本以及 `msedgedriver.exe`。

### 配置文件 (`config.json`)

请按照以下步骤配置：

1. 复制目录下的 `config.example.json` 并重命名为 `config.json`。
2. 修改 `config.json` 填写你的实际信息（山理用户仅需改动 3 处）：
* **url**: 山东理工大学教务系统实际登录地址。
* **student_number**: 你的山理学号。
* **password**: 你的教务系统密码。



---

## 🚀 4. 使用方法

### 运行脚本

在已激活虚拟环境的终端中输入：

```bash
python main.py
```

### 运行流程

1. 程序会自动启动 Edge 浏览器（默认为无界面模式，在后台运行）。
2. 自动跳转登录页面，输入学号密码。
3. 自动跳转选课页面。
4. 脚本会按照设定好的逻辑尝试抢课。
5. 程序运行结束后，在终端界面按回车键关闭浏览器。

---

## 💡 5. 注意事项与优化

* **反爬虫**: 如果学校系统有图形验证码，此脚本需要进一步开发识别验证码的功能。
* **速度瓶颈**: Selenium 速度较慢。如需追求极速，建议深入研究 `requests` 库直接发送 HTTP POST 请求。
* **iframe 切换**: 正方系统大量使用 `iframe`，脚本中已包含自动切换逻辑，如遇选课失败，请检查选课页面的 iframe 结构是否变动。
